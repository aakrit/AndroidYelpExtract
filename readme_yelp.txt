To use the Yelp API, you must first complete the following steps:

1) sign up for Yelp and log in

2) request an API key at http://www.yelp.com/developers/getting_started/api_access

Then you are all set to use the Yelp API.

Some helpful links from the Yelp site:
Documentation: (http://www.yelp.com/developers/documentation/v2/overview)
Basic examples: (https://github.com/Yelp/yelp-api/tree/master/v2/)
